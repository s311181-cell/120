# MINEJOURNAL ✦ 我的星光旅程

一个现代化的演唱会日志追踪应用，帮助你记录、管理和分析你的追星之旅。

## 🌟 功能特性

- **演唱会记录管理** - 记录你参加的每一场演唱会
- **Firebase 认证** - 安全的用户登入/注册系统
- **响应式设计** - 完美适配手机和桌面设备
- **深色主题** - 护眼的现代 UI 设计
- **数据持久化** - 云端安全存储你的数据

## 🚀 快速开始

### 1. 打开应用
直接在浏览器中打开 `index.html` 文件，或访问部署的在线版本。

### 2. 注册账户
点击"注册"选项卡，输入你的邮箱和密码来创建新账户。

### 3. 开始记录
登入后，你可以开始记录你的演唱会经历。

## 🔧 技术栈

- **前端框架**: 纯 HTML5 + CSS3 + JavaScript
- **后端服务**: Firebase (Authentication & Firestore)
- **字体**: Google Fonts (Cormorant Garamond, Noto Sans TC, Space Mono)
- **样式方案**: CSS Custom Properties (CSS 变量)

## 🎨 设计特点

- **玻璃态效果** (Glassmorphism) - 现代视觉效果
- **响应式布局** - 移动优先设计
- **黑暗主题** - 减少眼睛疲劳
- **流畅动画** - 增强用户体验

## 🔐 隐私与安全

- 所有数据存储在 Firebase 上
- 使用 Firebase Authentication 进行安全认证
- 会话级持久化 - 登出时数据会清除
- 密码从不以明文形式存储
